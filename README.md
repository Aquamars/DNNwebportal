# DNN webportal #



### What is this repository for? ###

* This portal for create instance to run DNN.
* Version 0.1.5

### How do I get set up? ###
```
npm start
```
* for development
```
npm run dll
npm run app
```
* package dll(bundle.js) and app(app.js)
* the file will generate in build folder
```
npm run build
```
* package all (dll and app)
* the file will generate in build folder
